## Auth



### 1. Sign Up



***Endpoint:***

```bash
Method: POST
Type: RAW
URL: http://localhost:5000/api/v1/auth/signup
```



***Body:***

{

    "password": "secret",
    "firstName": "Omar",

    "phone":"01112345566",
